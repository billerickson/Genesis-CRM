<h2>Instructions</h2>

<strong>Menu.</strong> To edit the site's menus, go to Appearance > Menus and select the appropriate menu at the top. <a href="http://en.support.wordpress.com/menus/">More information on Menus.</a>

<strong>Pages.</strong> To edit a page's content, go to Pages and select the appropriate page. You can insert an image using the first icon next to Upload/Insert. <a href="http://en.support.wordpress.com/pages/">More information on Pages.</a>

<strong>Posts.</strong> To edit the posts in your blog, go to the Posts section. <a href="http://en.support.wordpress.com/posts/">More information on Posts.</a>

<strong>Sidebar.</strong> Customize the sidebar widgets from Appearance > Widgets. <a href="http://en.support.wordpress.com/widgets/">More information on Sidebars.</a>
